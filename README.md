# 中文自然语言处理实验

这是一个中文自然语言处理实验仓库，主要包含中文分词、词性标注和命名实体识别相关的实验代码与数据。

## 项目内容

项目包含两类实验：

- 基于 HMM 的中文分词与词性标注实验。
- 面向 CRF++ 格式的数据预处理与命名实体识别评估脚本。

## 文件说明

| 文件 | 说明 |
| --- | --- |
| `hmm.py` | HMM 实验脚本，包含数据清洗、字典构建、HMM 参数统计、Viterbi 解码、分词与词性标注评估逻辑。 |
| `crf.py` | CRF 数据处理与结果评估脚本，用于生成 CRF++ 风格训练/测试数据，并对标注结果做简单统计。 |
| `raw_data.txt` | 带词性标注的中文语料，用于 HMM 分词与词性标注实验。 |
| `data_nr.txt` | 命名实体识别相关语料，用于 CRF 数据预处理。 |
| `template` | CRF++ 特征模板文件。 |

## 生成文件

脚本运行过程中可能生成分词结果、词性标注结果、临时语料和模型文件。这些文件不属于项目源文件，已通过 `.gitignore` 忽略。
